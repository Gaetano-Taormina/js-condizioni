# Esercizio

* Chiedi username e password. Se sono rispettivamente "admin" e "1234", stampa "Login effettuato", altrimenti stampa "Credenziali errate".

## Ragionamento

* leggi username e password
* se username == admin && password == "1234" stampa Login effettuato
  * altrimenti stampa Credenziale errate

# Exercise

* Ask username and password. If they are respectively "admin" and "1234", print "Login successful", otherwise print "Invalid credentials".

## Reasoning

* read username and password
* if username == admin && password == "1234" print Login successful
  * otherwise print Invalid credentials
