# Esercizio

* Gaetano:
Chiedi all'utente di inserire il suo nome. Verifica che non abbia lasciato il campo vuoto.

## Risoluzione

* dichiarare variabile nome
* leggi il nome dell utente
  * se l'utente non ha inserito il nome dell'utente
    * farlo notare all utente
* tornare indietro in caso di mancanza nome
* dare come risultato il nome dell utente
