# Esercizio
*   Chiedi l'età dell'utente tramite prompt. Se l'utente è maggiorenne, stampa a console "Accesso consentito al locale".

## Ragionamento

* leggi l'eta dell'utente 
* if utente maggiorenne stampa messaggio
* else windows.open(''.'self') per aprire e window.close() per chiudere la finestra ai minorenni
