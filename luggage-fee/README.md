# Exercise

- The ticket cost is 36 €. Ask the luggage weight in kg. If the weight exceeds 23kg, there is a 20 € surcharge, finally print the ticket price.

## Reasoning

- the ticket cost varies only under a certain condition
- the luggage weight can vary
- ask the luggage weight
- read the luggage weight
  - if luggage weight exceeds 23 kg add 20 euros to the ticket price
