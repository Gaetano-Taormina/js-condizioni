# Esercizio

* Chiedi un voto da 1 a 10. Stampa "Insufficiente" se < 6, "Sufficiente" se tra 6 e 7, "Buono" se tra 8 e 9, "Ottimo" se 10.

## Ragionamento

* leggere un valore
* controllare se dato valore è tra 1 e 10

* se n <  6 stampa insufficiente
  * altrimenti se n < 8 stampa sufficiente
    * altrimenti se n < 10 stampa buono
      * altrimenti  stampa ottimo

---

## Exercise

* Ask for a grade from 1 to 10. Print "Insufficient" if < 6, "Sufficient" if between 6 and 7, "Good" if between 8 and 9, "Excellent" if 10.

## Reasoning

* read a value
* check if given value is between 1 and 10

* if n < 6 print insufficient
  * otherwise if n < 8 print sufficient
    * otherwise if n < 10 print good
      * otherwise print excellent
