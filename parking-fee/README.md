# Esercizio

Chiedi quante ore si resta nel parcheggio. Se sono <= 2 ore costa 5€, se sono <= 5 ore costa 10€, altrimenti costa 15€.

## Ragionamento

* leggi ore
* se ore > 5 stampa costo 15 euro
  * altrimenti se ore > 2  stampa costo 10
  * altrimenti costo 5

---

# Exercise

Ask how many hours you stay in the parking lot. If <= 2 hours it costs 5€, if <= 5 hours it costs 10€, otherwise it costs 15€.

## Reasoning

* read hours
* if hours > 5 print cost 15 euros
  * otherwise if hours > 2 print cost 10
  * otherwise cost 5
